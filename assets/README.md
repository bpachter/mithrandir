# Enkidu — Visual Assets

Drop GIFs and short clips here. Filenames are referenced from the project READMEs and `LINKEDIN_POST.md`.

## Conventions

- **Format**
  - Short loops → `.gif` (≤ 8 s, ≤ 5 MB) for inline README use
  - Longer demos → `.mp4` (H.264, ≤ 30 s) — link only, GitHub does not auto-render mp4 inline
- **Sizing**
  - Inline README images: 1280 px wide max
  - LinkedIn carousel slides: 1200 × 1200 px (square) or 1200 × 627 px (landscape)
  - GIFs: 960 px wide max to keep file size manageable
- **Naming** — kebab-case, phase-prefixed: `phase1-ollama-pull.gif`, `phase6-ui-hero.png`, `phase3-telegram-react-loop.gif`
- **Privacy** — blur API keys, Telegram usernames, and any personally identifiable filesystem paths before committing
